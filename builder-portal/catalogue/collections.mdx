Collections allow you to group related catalogue items together. Use collections to create curated sets of items for specific purposes, styles, or project types, and to enable quick option‑switching in proposals and selections.

## What are collections?

Collections are organized groups of catalogue items. Examples include:

- **Fixture packages** — Matching bathroom or kitchen fixtures
- **Style bundles** — Items that fit a design aesthetic
- **Standard specifications** — Your default selections
- **Premium upgrades** — Higher-end alternatives
- **Allowance packages** — Pre-set allowance items

## Collection list

View all collections with:

| Column | Description |
|--------|-------------|
| **Name** | Collection name |
| **Items** | Number of items |
| **Description** | Collection purpose |
| **Status** | Active or archived |

## Creating collections

To create a new collection:

1. Click **New Collection**
2. Enter collection details:
   - Name
   - Description
   - Category (optional)
3. Add items to the collection
4. Save

## Adding items

Add items to a collection:

1. Open the collection
2. Click **Add Items**
3. Search or browse catalogue items
4. Select items to add
5. Save

## Using collections

Collections can be used to:

- Quickly populate selections
- Create proposal packages
- Set standard specifications
- Offer upgrade packages

## Editing collections

Modify collections at any time:

- Add or remove items
- Update name and description
- Reorder items
- Adjust pricing overrides

<Tip>
Create collections for your most common project types. This speeds up creating new projects with pre-selected items.
</Tip>

## Collection organization

Organize collections by:

- Project type (new build, renovation)
- Style (modern, traditional)
- Price point (standard, premium)
- Category (bathroom, kitchen)

## Archiving collections

When a collection is no longer needed:

1. Open the collection
2. Click **Archive**
3. Confirm

<Note>
Archiving a collection doesn't affect the individual items within it. Items remain in your catalogue.
</Note>
