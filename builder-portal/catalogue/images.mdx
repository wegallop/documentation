The Images section is your centralized image library. Store, organize, and reuse images across proposals, selections, the client portal, and project documentation.

## Image library

View all images with:

- Thumbnail preview
- Image name
- Tags
- Upload date
- File size

## Uploading images

To add new images:

1. Click **Upload**
2. Select images from your device
3. Add names and tags
4. Save

Supported formats:

- JPEG/JPG
- PNG
- GIF
- WebP

## Image organization

### Tags

Tag images for easy searching:

- Product type (faucet, tile, flooring)
- Style (modern, traditional, transitional)
- Brand name
- Project name
- Color/material

### Folders

Some organizations use folder structures:

- By product category
- By manufacturer
- By project type
- By style

## Using images

Insert library images into:

- **Selections** — Option images
- **Proposals** — Product photos
- **Catalogue items** — Product images
- **Project files** — Reference images

## Image details

Click on any image to view:

- Full-size preview
- Image name and description
- Tags
- File information
- Usage history

## Editing image information

Update image metadata:

1. Click on the image
2. Edit name, description, or tags
3. Save changes

<Note>
You cannot replace an image file after upload. To update an image, upload the new version and archive the old one.
</Note>

## Searching images

Find images using:

- Text search (name, description)
- Tag filters
- Date filters

<Tip>
Develop a consistent tagging system for your images. Good organization makes finding the right image much faster.
</Tip>

## Best practices

### Image quality

- Use high-resolution images
- Consistent aspect ratios when possible
- Professional product photos
- Good lighting and staging

### Naming conventions

- Descriptive names
- Include brand or product line
- Use consistent formatting
