The Items section is your master product and services library. Each item behaves like a reusable building block for proposal Item blocks and selection options, with consistent descriptions, pricing, and coding.

## Item list

View all items with:

| Column | Description |
|--------|-------------|
| **Name** | Item name |
| **Category** | Item category |
| **Price** | Standard price |
| **Unit** | Pricing unit (each, sq ft, etc.) |
| **Status** | Active or archived |

## Creating items

To add a new item:

1. Click **New Item**
2. Enter item details:
   - Name
   - Description
   - Category
   - Pricing information
   - Images
3. Save the item

## Item details

Each item can include:

### Basic information

- **Name** — Display name
- **Description** — Detailed description
- **Category** — Organization category
- **Tags** — Searchable tags

### Pricing

- **Price** — Standard unit price.
- **Unit** — How the item is priced (each, sq ft, hour, etc.).
- **Cost** — Your internal cost (for margin tracking).
- **Budget type** — Whether the item is typically fixed‑price or allowance‑based.

### Media

- **Images** — Product photos
- **Documents** — Spec sheets, PDFs

## Item categories

Organize items into categories:

- Fixtures
- Finishes
- Appliances
- Materials
- Labor
- Services
- Allowances

## Using items

Insert catalogue items into:

- **Proposals** — Add as line items
- **Selections** — Create selection options
- **Change orders** — Add to modifications
- **Allowances** — Set as allowance items

## Editing items

Update items at any time:

1. Click on the item
2. Modify the desired fields
3. Save changes

<Note>
Changes to catalogue items don't automatically update documents that already use them. This protects existing proposals and contracts.
</Note>

## Archiving items

When items are no longer used:

1. Open the item
2. Click **Archive**
3. Confirm

Archived items are hidden but preserved for historical reference.

<Tip>
Review your item catalogue periodically to update pricing and archive discontinued products.
</Tip>

## Searching and filtering

Find items using:

- Search by name or description
- Filter by category
- Filter by tags
- Sort by various fields
