The Text section stores reusable text content for your organization. Create standard descriptions, notes, and explanations that can be inserted into documents consistently and linked to proposal Text blocks.

## Text library

View all text blocks with:

| Column | Description |
|--------|-------------|
| **Name** | Text block name |
| **Category** | Type of content |
| **Preview** | First few lines |
| **Status** | Active or archived |

## Types of text content

### Scope descriptions

Standard descriptions of work:

- Project phases
- Trade scopes
- Service descriptions
- Deliverables

### Notes and disclaimers

Important notices:

- Allowance disclaimers
- Site condition notes
- Timeline caveats
- Pricing notes

### Process explanations

Client-facing content:

- How selections work
- Payment process
- Change order process
- Communication guidelines

### Marketing content

Promotional text:

- Company descriptions
- Service overviews
- Differentiators
- Testimonials

## Creating text blocks

To add new text:

1. Click **New Text**
2. Enter content:
   - Name (internal reference)
   - Category
   - Full text content
3. Save

## Formatting

Text blocks support:

- Paragraphs
- Bullet points
- Numbered lists
- Bold and italic
- Headings

## Using text

Insert text blocks into:

- Proposals and estimates
- Selections
- Emails and messages
- Project documentation

<Tip>
Create text blocks for any content you find yourself typing repeatedly. Consistency improves professionalism and saves time.
</Tip>

## Editing text

Update text blocks as needed:

1. Open the text block
2. Modify the content
3. Save changes

<Note>
Changes don't update documents that already include the text. This preserves the content as it was when the document was created.
</Note>

## Best practices

### Clarity

- Write clearly and concisely
- Avoid jargon with clients
- Use active voice

### Consistency

- Maintain consistent tone
- Use standard terminology
- Follow style guidelines

### Organization

- Meaningful names
- Logical categories
- Good search tags

## Archiving text

When text is no longer needed:

1. Open the text block
2. Click **Archive**
3. Confirm

Archived text remains available for reference.
