The Contracting section provides an organization-wide view of all proposals and change orders. Track the status of contracts and changes across all your projects from one location.

## Overview

The Contracting section gives you visibility into:

- All proposals across projects
- Change order status and approvals
- Contract values and totals
- Pending signatures

## Navigation

Use the tabs to switch between:

- **Proposals** — View all proposals
- **Change Orders** — View all change orders

## Quick actions

From this section, you can:

- View proposal details
- Check approval status
- Track signature progress
- See contract values
- Follow up on pending items

<Tip>
Use the Contracting section for a high-level view of your pipeline. For detailed editing, work within individual projects.
</Tip>

## Filtering

Filter contracting items by:

- **Project** — Show items from specific projects
- **Status** — Draft, sent, signed, etc.
- **Preparer** — Who created the item
- **Date range** — When created or sent

<Note>
Creating new proposals and change orders is done within individual projects. This section provides an aggregate view across your organization.
</Note>
