The Financial Dashboard provides a high-level view of your organization's financial health. Monitor key metrics and identify areas requiring attention.

## Dashboard overview

The dashboard displays key financial information:

### Revenue metrics

- **Total contract value** — Sum of all active contracts
- **Invoiced to date** — Total amount billed
- **Payments received** — Total payments collected
- **Outstanding balance** — Amount still owed

### Cash flow

Visual representation of:

- Incoming payments
- Outgoing costs
- Net cash position

### Receivables aging

Track how long invoices have been outstanding:

| Category | Description |
|----------|-------------|
| **Current** | Not yet due |
| **1-30 days** | Recently overdue |
| **31-60 days** | Moderately overdue |
| **60+ days** | Significantly overdue |

## Key insights

The dashboard highlights:

- Projects with overdue payments
- Upcoming large payments
- Projects over budget
- Unusual activity

## Quick actions

From the dashboard, quickly:

- View projects with outstanding balances
- See upcoming scheduled payments
- Access overdue invoices
- Review recent transactions

<Tip>
Review the financial dashboard weekly to stay on top of cash flow and identify any issues early.
</Tip>

## Filtering

Narrow the dashboard view by:

- Date range
- Project
- Client
- Transaction type

## Exporting data

Export financial data for external reporting:

- Download as CSV or Excel
- Generate PDF reports
- Schedule automatic reports

<Note>
Dashboard data is real-time. Values update automatically as transactions are recorded.
</Note>
