The Client Transactions section shows all financial transactions with clients across your organization. Track invoices, payments, and account activity.

## Transaction list

View all transactions with:

| Column | Description |
|--------|-------------|
| **Date** | Transaction date |
| **Project** | Associated project |
| **Client** | Client name |
| **Type** | Invoice, payment, credit |
| **Amount** | Transaction value |
| **Status** | Current status |

## Transaction types

### Invoices

Bills sent to clients for work performed:

- Progress invoices
- Milestone invoices
- Change order invoices
- Final invoices

### Payments

Funds received from clients:

- Check payments
- Bank transfers
- Online payments
- Credit card payments

### Credits

Adjustments to client accounts:

- Allowance credits
- Refunds
- Adjustments

## Filtering transactions

Filter by:

- **Project** — Specific project
- **Client** — Specific client
- **Type** — Invoice, payment, or credit
- **Date range** — Specific time period
- **Status** — Paid, pending, overdue

## Searching

Find transactions by:

- Invoice number
- Project name
- Client name
- Amount

## Transaction details

Click on any transaction to view:

- Full transaction details
- Line items (for invoices)
- Payment method (for payments)
- Related items
- Notes and attachments

## Recording payments

When a payment is received:

1. Find the related invoice
2. Click **Record Payment**
3. Enter payment details
4. Select payment method
5. Save the transaction

<Tip>
Record payments promptly to keep your financial data accurate and your dashboard current.
</Tip>

## Reconciliation

Use this section to:

- Match payments to invoices
- Identify unallocated payments
- Resolve discrepancies
- Track partial payments

<Note>
If you use accounting software, transactions can be synced through integrations. See the Settings section for integration options.
</Note>
