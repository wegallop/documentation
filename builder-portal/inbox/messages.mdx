The Messages section in your Inbox provides a unified view of all conversations happening across your projects. It brings together project chats, direct messages, and decision‑linked threads in one place so you can stay connected without switching between projects.

## Message list

The message list shows all your conversations with:

- **Conversation name** — Project chat name or participant names
- **Project** — Which project the conversation belongs to
- **Last message** — Preview of the most recent message
- **Time** — When the last message was sent
- **Unread indicator** — Shows if there are unread messages

## Conversation types

### Project chats

Main communication channels for each project. Visible to all assigned team members.

### Direct messages

One-on-one conversations with clients or team members. Private to the participants.

### Decision threads

Conversations linked to specific decisions, selections, or tasks. Keeps discussion contextual.

## Reading conversations

Click on any conversation to open it:

- Full message history is displayed
- Scroll to view older messages
- New messages appear at the bottom
- Context link shown for decision threads

## Sending messages

To send a message:

1. Click on the conversation
2. Type your message in the text field
3. Press Enter or click Send

### Attachments

Share files in conversations:

1. Click the attachment icon
2. Select files from your device
3. Add optional message text
4. Send

## Starting new conversations

To start a new conversation:

1. Click **New Message**
2. Select the project
3. Choose participants
4. Type your first message
5. Send

<Tip>
Use @mentions to notify specific team members in project chats. They'll receive a notification even if they've muted the conversation.
</Tip>

## Managing conversations

### Mark as read/unread

Keep track of conversations you need to revisit by marking them as unread.

### Mute conversations

Mute busy conversations to reduce notification noise. You can still view them but won't receive alerts.

## Searching messages

Use the search function to find specific conversations or messages:

- Search by keyword
- Filter by project
- Filter by participant
- Search within date ranges

<Note>
All messages are archived and searchable, making it easy to reference past discussions.
</Note>
