The Tasks section in your Inbox provides a consolidated view of all tasks from across your projects. It includes internal tasks, client tasks you are responsible for, and workflow items generated by other Gallop features such as proposals, change orders, and requests. Stay organized and ensure nothing falls through the cracks.

## Task views

Switch between different views to focus on what matters:

### My Tasks

Tasks assigned to you that need completion. This is your personal to‑do list across all projects.

### Assigned by Me

Tasks you've delegated to others. Track progress and follow up on outstanding items.

### Team Tasks

All tasks across your team. Useful for managers to monitor workload and progress.

## Task list

Each task displays:

| Field | Description |
|-------|-------------|
| **Title** | Task name and short description |
| **Project** | Which project the task belongs to (if applicable) |
| **Source** | Where the task came from (for example, Proposal review, Change request, Client Task) |
| **Assignee** | Who is responsible |
| **Due date** | When the task is due (if applicable) |
| **Status** | Current task status |
| **Priority** | Task priority level |

## Task status

Tasks move through these statuses:

| Status | Description |
|--------|-------------|
| **Open** | Task is pending action |
| **In Progress** | Work has started |
| **Completed** | Task is done |
| **Overdue** | Past the due date |

## Filtering tasks

Use filters to focus on specific tasks:

- **Status** — Open, in progress, completed, overdue
- **Project** — Tasks from specific projects
- **Assignee** — Tasks for specific team members
- **Due date** — Tasks due within a time range
- **Priority** — High, medium, low priority

## Working with tasks

### Viewing task details

Click on any task to see:

- Full description.
- Checklist items and completion percentage (if present).
- Attached files.
- Comments and activity.
- Related items (for example, selections, proposals, change orders, or requests).

### Completing tasks

When you finish a task:

1. Open the task.
2. Review the requirements and any checklist items.
3. Add any necessary response or notes.
4. Mark the task (and checklist, if present) as complete.

### Reassigning tasks

If a task should be handled by someone else:

1. Open the task.
2. Click on the assignee.
3. Select the new team member.
4. Save the change.

<Tip>
Review overdue tasks first thing each day. Quickly reassign or complete them to keep projects moving.
</Tip>

## Task notifications

You'll receive notifications when:

- A task is assigned to you
- A task you're watching is updated
- A task is approaching its due date
- A task becomes overdue

<Note>
Configure task notification preferences in your account settings.
</Note>
