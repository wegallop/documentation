Each project has its own settings that control how it operates, who has access, and how features are configured. Access project settings from within any project.

## Settings sections

### Overview

Basic project information:

- **Project name** — Display name for the project
- **Address** — Project location
- **Project type** — Type of construction (new build, renovation, etc.)
- **Stage** — Current project stage
- **Description** — Project notes and description
- **Image** — Project photo or rendering

### Team

Manage who works on the project:

- Add or remove team members
- Assign roles and responsibilities
- Set project-specific permissions
- Designate primary contacts

### Clients

Control client access:

- Add clients to the project
- Set client permissions
- Control what sections clients can view
- Manage client invitations

### Security

Access and visibility controls:

- **Portal access** — Enable/disable client portal
- **Section visibility** — Control which sections clients see
- **File permissions** — Upload and view permissions
- **Schedule visibility** — What schedule details are shared

### Payment

Payment configuration:

- Payment terms
- Payment schedules
- Deposit requirements
- Late payment policies

### Requests

Change request settings:

- Enable change requests
- Request approval workflow
- Notification settings
- Required fields

### Warranty

Warranty configuration:

- Warranty period duration
- Coverage categories
- Request submission settings
- Response requirements

### Financial

Financial setup and management:

#### Budget
- Set project budgets
- Budget categories
- Tracking and alerts

#### Cost reports
- Generate cost reports
- Track actuals vs. budget
- Variance analysis

#### Costs forecasts
- Project future costs
- Cash flow planning
- Budget projections

#### Settings
- Tax configuration
- Currency settings
- Financial permissions

## Permission levels

Different settings require different permission levels:

| Level | Can modify |
|-------|------------|
| **Project member** | Limited settings |
| **Project manager** | Most project settings |
| **Admin** | All settings including financial |

<Tip>
Set up project templates with pre-configured settings to streamline new project creation.
</Tip>

## Saving changes

Changes to project settings are saved immediately. Some changes may require:

- Confirmation for sensitive settings
- Re-authentication for security changes
- Notification to affected team members
