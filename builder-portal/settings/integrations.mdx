Integrations connect Gallop with external services. Sync data with accounting software, payment processors, and other business tools so you can reduce manual entry and keep financial information aligned.

<Note>
Integration settings are only accessible to administrators.
</Note>

## Available integrations

### QuickBooks Online

Sync financial data with QuickBooks Online:

#### Setup

1. Click **Connect to QuickBooks**
2. Sign in to your QuickBooks account
3. Authorize the connection
4. Configure sync settings

#### Features

- **Cost code ↔ Products/services mapping** — Align Gallop cost codes with QuickBooks products and services.
- **Project ↔ Customer/project mapping** — Map Gallop projects to QuickBooks customers and sub‑jobs.
- **Transaction sync** — Sync invoices, credit memos, refunds, and payment receipts.
- **Purchase order import** — Bring purchase order data into Gallop for cost reporting.

#### Configuration options

| Setting | Description |
|---------|-------------|
| **Account** | Connected QuickBooks company |
| **Projects** | How projects sync |
| **Cost codes** | Cost code mapping rules |
| **Rules** | Automation rules |
| **Transactions** | Sync settings |
| **Invoice accounts** | Revenue account mapping |

### Other integrations

Additional integrations may be available:

- Payment processors
- File storage
- Communication tools
- Scheduling software

## Setting up integrations

General process:

1. Navigate to the integration
2. Click **Connect** or **Set up**
3. Authenticate with the service
4. Configure mapping and options
5. Test the connection
6. Enable sync

## Managing integrations

### Connection status

View integration health:

- Connected/disconnected
- Last sync time
- Error status

### Sync settings

Control how data flows:

- Automatic vs. manual sync
- Sync frequency
- What data syncs
- Direction (one-way or two-way)

### Error handling

When sync issues occur:

- View error details
- Retry failed syncs
- Fix mapping issues
- Contact support if needed

<Tip>
Test integrations thoroughly before relying on them. Verify data is syncing correctly before going live.
</Tip>

## Disconnecting integrations

To remove an integration:

1. Open integration settings
2. Click **Disconnect**
3. Confirm the action

<Note>
Disconnecting an integration stops data sync but doesn't delete historical data in either system.
</Note>
