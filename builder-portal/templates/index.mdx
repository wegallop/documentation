## Why use templates?

Templates provide significant benefits:

- **Speed** — Create documents quickly
- **Consistency** — Standard format and content
- **Quality** — Pre-approved content and pricing
- **Training** — New team members follow proven patterns

## Getting started

### Analyze your workflow

Identify documents you create frequently:

- What proposals do you send most often?
- What selections do clients typically make?
- What tasks recur on every project?

### Build base templates

Start with your most common scenarios:

- Standard proposal structure
- Typical selection categories
- Common task lists

### Refine over time

Improve templates based on:

- Client feedback
- Team input
- Process improvements
- New offerings

<Tip>
Create multiple template versions for different project types. A renovation proposal template will differ from new construction.
</Tip>

## Template management

For each template type, you can:

- Create new templates
- Edit existing templates
- Duplicate templates
- Archive unused templates

## Using templates

When creating new documents:

1. Select "Create from template"
2. Choose the appropriate template
3. Customize for the specific project
4. Complete and send

<Note>
Templates are starting points. Always review and customize content for each specific project and client.
</Note>
