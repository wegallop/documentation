Selection templates define the choices clients need to make on their projects. Pre-build selection categories and options to streamline the decision-making process.

## Template list

View all selection templates:

| Column | Description |
|--------|-------------|
| **Name** | Template name |
| **Category** | Selection category |
| **Options** | Number of options |

## Creating templates

To create a new selection template:

1. Click **New Template**
2. Define the selection:
   - Name
   - Category
   - Description
   - Instructions for client
3. Add options:
   - Option names
   - Descriptions
   - Images
   - Pricing
4. Save the template

## Template structure

### Selection information

- **Name** — What the client is selecting
- **Category** — Grouping (plumbing, flooring, etc.)
- **Description** — Context and guidance
- **Due date** — When selection is needed

### Options

Each option includes:

- **Name** — Option identifier
- **Description** — Details about this choice
- **Images** — Visual representation
- **Price** — Cost or allowance impact
- **Specifications** — Technical details

Create selection templates with your standard options and pricing. Add project-specific options when needed.
</Tip>

## Using templates

When adding selections to a project:

1. Choose from template
2. Customize options if needed
3. Set due date
4. Assign to client
5. Publish selection

## Organizing options

Within each selection:

- Order options by popularity
- Lead with recommended choice
- Include budget and premium options
- Show clear pricing differences

<Note>
Keep templates updated with current products and pricing. Outdated information creates confusion.
</Note>

## Best practices

- **Clear images** — Use high-quality product photos
- **Complete descriptions** — Include all relevant details
- **Accurate pricing** — Reflect current costs
- **Good organization** — Logical option order
