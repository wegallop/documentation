The Files section provides access to all documents, images, and files associated with your project. Everything is organized and searchable so you can quickly find what you need, and many files are linked directly from other parts of the app.

## File organization

Files can be viewed and filtered in several ways:

### By source

- **Direct uploads** — Files uploaded directly to the project.
- **Schedule** — Files attached to schedule items.
- **Messages** — Files shared in conversations.
- **Selections** — Images and documents from selection options.
- **Tasks** — Files attached to task responses.
- **Requests** — Files attached to change or warranty requests.
- **Agreements** — Contracts, proposals, and signed documents.

### By type

- **Images** — Photos and graphics
- **Videos** — Video files
- **PDFs** — PDF documents
- **Spreadsheets** — Excel and other spreadsheet files
- **Documents** — Word documents and text files

## Browsing files

The files section offers two view modes:

| View | Description |
|------|-------------|
| **Grid view** | Thumbnail preview of files |
| **List view** | Detailed list with metadata |

## Searching files

Use the search bar to find files by:

- File name
- Description
- Tags
- Date range

## File details

Click on any file to view:

- Full preview (for supported formats)
- File name and description
- Upload date
- Source (where the file came from)
- File size
- Tags and metadata

## Uploading files

Depending on your permissions, you may be able to upload files directly to the project:

1. Click the **Upload** button.
2. Select files from your device.
3. Add a description (optional).
4. Click **Upload**.

<Tip>
Use descriptive file names to make files easier to find later. For example, "Kitchen-tile-option-A.jpg" is more helpful than "IMG_1234.jpg".
</Tip>

## Downloading files

To download files:

1. Click on the file to open it
2. Click the **Download** button
3. The file will save to your device

You can also download multiple files at once by selecting them and using the bulk download option.

## Common file types

| Extension | Type | Opens with |
|-----------|------|------------|
| .pdf | PDF document | PDF viewer |
| .jpg, .png | Image | Image viewer |
| .mp4 | Video | Video player |
| .docx | Word document | Word processor |
| .xlsx | Spreadsheet | Spreadsheet app |

<Note>
File upload permissions may be restricted based on your project settings. Contact your team if you need to share files but don't see the upload option.
</Note>

## Project documentation

Important project files you might find here:

- **Plans and drawings** — Architectural and construction drawings.
- **Specifications** — Product specifications and cut sheets.
- **Photos** — Progress photos and site documentation.
- **Contracts and proposals** — Signed agreements, proposals, and change orders.
- **Reports** — Inspection reports, briefings, and certifications.
