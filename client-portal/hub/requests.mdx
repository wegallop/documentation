The Requests section allows you to submit and track change requests for your project. A change request is a formal way to propose modifications to your project scope, design, or specifications.

## What is a change request?

A change request is used when you want to:

- Modify the original design or plans.
- Add new features or elements.
- Remove items from the scope.
- Change materials or finishes.
- Adjust timelines or priorities.

## Viewing requests

All your change requests are displayed in a list view showing:

- Request title and description
- Current status
- Date submitted
- Associated cost impact (if any)

## Submitting a change request

To submit a new change request:

1. Click the **New Request** button
2. Enter a clear title for your request
3. Provide a detailed description of the change you're requesting
4. Attach any relevant files or images
5. Submit the request

<Tip>
Be as specific as possible when describing your change request. Include details like dimensions, colors, or reference images to help your team understand exactly what you're looking for.
</Tip>

## Request status

Change requests move through several stages:

| Status | Description |
|--------|-------------|
| **Submitted** | Your request has been received |
| **Under Review** | Your team is evaluating the request |
| **Quoted** | A cost estimate has been provided |
| **Approved** | The change has been approved and will proceed |
| **Declined** | The request was not approved |
| **Completed** | The change has been implemented |

## Tracking progress

Once a change request is submitted, you can:

- View the current status
- See any comments or questions from your team
- Review cost estimates when available
- Track when the change is implemented

<Note>
Change requests may impact your project timeline and budget. Your team will provide detailed information about any impacts before proceeding.
</Note>
