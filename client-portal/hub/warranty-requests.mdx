The Warranty Requests section allows you to submit and track warranty claims for issues that arise after your project is complete. This ensures any covered repairs or corrections are properly documented and addressed.

## What is covered?

Warranty coverage varies by project and contract. Common warranty items include:

- Structural defects.
- Workmanship issues.
- Material failures.
- System malfunctions (HVAC, plumbing, electrical).
- Finish defects.

<Note>
Refer to your project contract or contact your team for specific warranty coverage details and duration.
</Note>

## Submitting a warranty request

To submit a new warranty request:

1. Click **New Warranty Request**
2. Select the category or area affected
3. Describe the issue in detail
4. Add photos showing the problem
5. Submit your request

<Tip>
Clear photos are extremely helpful for warranty requests. Take multiple angles and include close-ups of the affected area.
</Tip>

## Request information

When submitting a request, provide:

- **Location** — Where in the property is the issue?
- **Description** — What is the problem?
- **When noticed** — When did you first see the issue?
- **Photos** — Visual documentation of the problem

## Tracking your requests

Each warranty request shows:

| Field | Description |
|-------|-------------|
| **Status** | Current state of the request |
| **Date submitted** | When you submitted the request |
| **Category** | Type of issue |
| **Description** | Summary of the problem |

## Warranty request status

Requests progress through these stages:

| Status | Description |
|--------|-------------|
| **Submitted** | Request received and logged |
| **Under Review** | Team is evaluating the issue |
| **Scheduled** | Repair has been scheduled |
| **In Progress** | Work is being completed |
| **Completed** | Issue has been resolved |
| **Denied** | Issue not covered under warranty |

## Communication

Your team may reach out with questions about your warranty request. You can respond directly through the messaging feature to provide additional information or schedule repair visits.
