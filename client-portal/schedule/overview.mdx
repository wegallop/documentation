The Schedule Overview provides a quick snapshot of what's happening on your project, organized by timeframe. It pulls in active phases, upcoming milestones, inspections, and other important events so you can see the big picture at a glance.

## Layout

The overview is organized into three columns:

- **This Week** — Items happening in the current week
- **This Month** — Items scheduled for the current month
- **Next Month** — Items coming up in the following month

## What you'll see

Each column displays relevant schedule items:

### Phases

Active and upcoming project phases, showing:
- Phase name
- Start and end dates
- Current status

### Milestones

Key project milestones, including:
- Milestone name
- Target date
- Completion status

### Inspections

Scheduled inspections showing:
- Inspection type
- Scheduled date
- Status (scheduled, passed, failed)

### Permits

Permit status and deadlines:
- Permit type
- Application date
- Approval status

## Weather widget

The overview may include a weather widget showing current conditions at your project site. This helps you understand how weather might impact the schedule.

<Tip>
Check the overview regularly to stay informed about what's happening on your project. It's the quickest way to see upcoming activities.
</Tip>

## Understanding status

Items are color-coded to help you quickly understand their status:

| Color | Meaning |
|-------|---------|
| **Green** | On track or completed |
| **Yellow** | Upcoming or in progress |
| **Red** | Overdue or requires attention |

<Note>
The Schedule Overview shows a simplified view. For the complete project timeline, visit the Critical Path section.
</Note>
