<Update label="March 2026" description="Docs & catalogue">
  **Documentation** — New Mintlify docs with Client Portal and Builder Portal sections.

  **Catalogue** — Pricing defaults and toggle for catalogue items.
</Update>

<Update label="February 2026" description="Signing, notifications & warranty">
  **Proposals** — Proposal signing task; builder signer can view proposal before signing. Proposal archiving.

  **Notifications** — Web push notifications with configurable settings; maintenance mode handling.

  **Cost reports** — In‑report chat for discussions.

  **Logs** — Scheduled logs; edit published logs.

  **Projects** — Project-level warranty settings; warranty and change request visibility when disabled. Role label updated from support/lead to member.
</Update>

<Update label="January 2026" description="Proposals & units">
  **Proposals** — Faster editing: inline cache updates, fewer refetches when adding, reordering, or deleting blocks.

  **Settings** — Project warranty settings view. Unit of measure replaced Area unit (configurable UoM).

  **Platform** — Next.js 16 upgrade.
</Update>

<Update label="December 2025" description="Permissions & team">
  **Team** — Team management and project role/permission switching.

  **Permissions** — Template, catalogue, and accounting permissions; project settings hidden when user lacks permission.
</Update>

<Update label="November 2025" description="Requests, logs & filters">
  **Requests** — Warranty requests and change requests with full tracking and notes.

  **Logs** — Logs support images and uploads; briefing image support.

  **Builder** — Cost code and category filtering; categorised cost codes in admin.
</Update>
